Task Manager

Сроки

2 месяца

О работе

Один из первых проектов на фреймворке Vue.js - Task manager. В данном проекте, мной был разработан фронтенд на Vue.js. В нём есть возможность создавать проект. Внутри проектов реализована возможность создания задачи, которая в свою очередь имеют статус, приоритет и тип. Также внутри проектов есть чаты, в котором пользователи могут обсуждать вопросы. Внутри задачи есть комментарии, где пользователи могут обсужать данную задачу. Присутствует возможность менять статус, приоритет и тип задачи, а также на кого она назначена.

Сайт сделан полностью динамически на Web сокетах. То есть, при изменении каких-либо данных, одним пользователем, находяющиеся на сайте остальные пользователи сразу увидят изменения, не прибегая к перезагрузке страницы.

Libertad

Сроки

3 месяца

О работе

Мой первый опыт разработки fullstack приложения. В основе данного приложения легла социальная сеть. Социальная сеть позволяет вести учёт просмотренных пользователем сериалов, выставлять оценки, делиться своим мнением, вести диалоги с другими пользователями.

Созданное мной клиент-серверное приложение включает в себя отдельное приложение на frontend, где используется Vue.js, которое получает данные через API сервера на Node.js.

На данном этапе нет возможности просмотра сайта. Но есть возможность посмотреть на сайт с технической стороны, на github.
